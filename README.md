# Investigating-Netflix-Movies

# Netflix 1990s Movie Analysis

## Project Overview
This project performs exploratory data analysis (EDA) on a Netflix dataset to uncover trends in movies released during the 1990s decade.

## Objectives
1.  **Filter Data**: Identify all movies released between 1990 and 1999.
2.  **Find Frequent Duration**: Determine the most common movie length (in minutes) during this period.
3.  **Count Short Action Movies**: Count how many 'Action' movies from the 1990s are shorter than 90 minutes.

## Findings
- **Most Frequent Duration**: 94 minutes.
- **Short Action Movies Count**: 7 movies.

## Dataset
The analysis uses `netflix_data.csv`, which includes information on titles, genres, release years, and durations.
